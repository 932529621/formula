### F1体育赛事网
######本系统分成前后台界面，游客可以进行赛事信息浏览，也可以注册成为网站会员，在登陆后收藏文章，参与讨论，使用支付宝购买赛事门票。会员需要实名认证后通过个人中心填写基础资料、身份认证及证件资料后申请成为车手，后台平台管理员审核通过后，通过前台信箱发放给车手后台登陆账号密码。
